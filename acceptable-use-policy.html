InArt Studio commits to full compliance with Amazon's Acceptable Use Policy for Selling Partner API usage.

Prohibited Activities - We Will NOT:

1. Access Unauthorized Data

  • Access data from seller accounts we don't manage
  • Attempt to circumvent API access controls
  • Use stolen or unauthorized credentials

2. Data Misuse

  • Sell, rent, or share Amazon data with third parties
  • Use buyer PII for unauthorized marketing
  • Scrape or cache data beyond permitted retention periods

3. System Abuse

  • Exceed API rate limits or quotas
  • Perform load testing without authorization
  • Attempt to reverse-engineer Amazon systems

4. Fraudulent Activities

  • Manipulate product reviews or ratings
  • Create fake accounts or impersonate others
  • Submit false information in developer registration

5. Security Violations

  • Store credentials insecurely
  • Share API access credentials
  • Fail to report security incidents within 24 hours

Our Commitments:

  • Use Amazon data only for authorized business purposes
  • Implement robust security controls
  • Maintain accurate developer profile information
  • Report security incidents promptly to security@amazon.com
  • Delete data when no longer needed
  • Comply with all applicable laws and regulations
  • Cooperate with Amazon audits and investigations

Enforcement

Violations of AUP may result in:

  • API access suspension or termination
  • Legal action by Amazon
  • Termination of seller account
